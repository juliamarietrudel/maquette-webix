# Les 9 services The WebiX

Les pages de service utilisent maintenant le même gabarit:
- `pages/service-template.html` pour rendre un service avec `?service=slug`
- `assets/js/service-template.js` pour le rendu
- `assets/css/service-template.css` pour le style partagé
- `assets/js/services-data.js` pour modifier le contenu de chaque service

Le header est aussi partagé:
- `assets/js/header.js` pour générer le header sur toutes les pages
- `assets/css/header.css` pour le style du header, basé sur la page Études de cas
